# movie-site-reference-library
本仓库用于整理和维护互联网上已公开可访问的影视类网站基础信息，主要面向电影与电视剧相关站点进行资料归档。记录内容包括站点名称、访问地址、站点定位说明及基础分类信息，方便后续查询、比对与长期留存。本项目不涉及任何影视资源的存储、播放或分发，仅对站点公开信息进行整理与记录。所有资料均来源于公开网络环境，整理过程遵循中立原则，不对站点内容质量、合法性或稳定性作出评价，也不构成任何形式的推荐或引导。
